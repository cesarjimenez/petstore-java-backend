Create complex pets

Given a pet with nested objects, the structure of the pet shall be maintained when the pet is retrieved.

Example

Given a pet ID pet_ID, when the pet 

{
  "id" : "pet_ID",
  "name" : "Toby",
  "owner" : "Me!",
  "group" : "Dogo",
  "type" : "Dog",
  "coords" : {
    "latitude" : 30,
    "longitude" : 0,
  },
  "info": {
    "height": 60,
    "color": "Black"
  }
}
is created, the nested object 
{height=60, color=Black}
shall be returned when the pet is retrieved.
